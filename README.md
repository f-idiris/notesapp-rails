### Technologies used

- Ruby on Rails 7
- Bootstrap 4
- HTML
- CSS
- SQLite

### Getting started

Clone the project onto your local machine:
```sh
git clone https://github.com/f-idiris/notesapp-rails.git
```
Install the dependencies:
```sh
bundle install
```
Setup the database:
```sh
rails db:create
rails db:migrate
```
Run the server:
```sh
rails server
```
Finally, open your browser and type ```localhost:3000```
